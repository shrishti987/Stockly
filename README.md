# 📊 Stockly – Smart Trading Dashboard
A modern, multi-feature stock market dashboard built using **React.js**, designed to simulate a real-world trading platform UI like Zerodha or Groww.

## 🚀 Features

### 🧩 Core Modules
- 📊 **Portfolio Analyzer**
  - Calculates total investment, current value, and P&L
  - Clean dashboard-style UI
- 📈 **Market Trends**
  - Displays stock trends with gain/loss indicators
  - Styled trading interface
- 🤖 **AI Insights**
  - Simple AI-based stock recommendation (Buy/Sell)
  - Futuristic UI panel
- 📰 **News & Updates**
  - Latest market news in a card-based feed
  - Scrollable and interactive UI
- 🔔 **Price Alerts**
  - Add and track stock alerts
  - Clean alert manager interface
- 🧮 **Brokerage Calculator**
  - Calculates brokerage charges
  - Fintech-style calculator UI
- 📅 **IPO Calendar**
  - Displays upcoming IPOs with status badges
  - Structured list UI
- 🏦 **Mutual Funds**
  - Shows top-performing funds
  - Investment-focused UI cards

## 🎨 UI Highlights
- 🌙 Modern **Dark Theme**

- 🧱 Modular **Component-Based Architecture**

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Styling:** CSS (custom styles)
- **Icons:** lucide-react
- **State Management:** React Hooks (`useState`)

## ⚙️ Installation & Setup
```bash
# Clone the repository
git clone https://github.com/your-username/stockly.git
# Navigate to project folder
cd stockly/dashboard
# Install dependencies
npm install
# Install icons
npm install lucide-react
# Start the app
npm start
